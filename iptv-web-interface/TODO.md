- 提供上下滑动切换频道的交互指示

- [ ] 收起时，播放面板自动扩展为全屏

### 2. 侧边栏搜索功能

**功能描述：**
- 在侧边栏添加搜索输入框
- 支持频道名称关键词搜索
- 实现模糊匹配功能

**实现要点：**
- [ ] 在侧边栏顶部添加搜索输入框
- [ ] 实现实时搜索（输入时即时过滤）
- [ ] 支持频道名称模糊匹配
- [ ] 支持频道 ID、国家等字段搜索（可选）
- [ ] 高亮显示匹配的关键词
- [ ] 显示搜索结果数量
- [ ] 清空搜索功能
- [ ] 搜索历史记录（可选）

**技术考虑：**
- 使用防抖（debounce）优化搜索性能
- 支持中文、英文等多语言搜索
- 考虑大数据量下的搜索性能优化
- 搜索结果按相关性排序

**搜索范围：**
- 频道名称（主要）
- 频道 ID（可选）
- 国家名称（可选）
- 频道分类（可选）
